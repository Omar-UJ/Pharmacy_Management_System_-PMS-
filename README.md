## Overview
The **Pharmacy Management System** is a modern, automated solution designed to enhance **accuracy, safety, and efficiency** in pharmaceutical stores.  
It automates inventory management, stock control, prescription handling, and reporting, helping pharmacists manage operations efficiently.

---

## Features
- Real-time inventory tracking
- Automated alerts for low stock and nearing expiry items
- Role-based access control
- Sales tracking and reporting
- Multi-user support
- User-friendly interface
- Reduced manual errors and paperwork

---

## Technologies Used
**Frontend:** React JS  
**Backend:** .NET API + Node.js API  
**Database:** Microsoft SQL Server  
**Languages:** Java, C#  
**Development Tools:** Visual Studio Code, NetBeans, XAMPP, Confluence  
**Hardware Requirements:** Intel Core i3, 4GB RAM, 256GB Disk Space
